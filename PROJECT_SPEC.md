# Ars Magna Scientiarum
## A Lullian Engine for the Discovery of Sciences

**Author:** R. Indira Valentina Réquiz Molina (Rainvare)

**Status:** Project Specification v1.0

## 1. Vision

A conversational system that applies Ramon Llull's combinatorial logic to the complete map of human knowledge, systematically generating hypotheses for new interdisciplinary fields, identifying unexplored territories in science, and validating suggestions against real bibliometric data.

**Core thesis:** The space of possible sciences is vastly larger than the space of existing sciences. A Lullian engine that combines the complete ontology of disciplines with typed operators and empirical validation can *cartograph the unknown* — revealing not just what exists, but what *could* exist and hasn't been tried.

## 2. Theoretical Foundation

### 2.1 From Llull to Science Discovery

Ramon Llull's *Ars Magna* (1305) proposed that all truths could be generated by systematically combining fundamental principles through defined operators. We preserve the mechanism but replace the domain:

| Ars Magna (Original) | Ars Magna Scientiarum |
|---|---|
| Principles (Bonitas, Magnitudo...) | Disciplines (complete scientific taxonomy) |
| Figures (combinatorial wheels) | Operators (Combinatio, Subordinatio, etc.) |
| Questions (Utrum, Quid, De quo...) | Generative queries on discipline intersections |
| Subjects (God, Angels, Man...) | Objects of study, methods, ontological assumptions |
| Truth generation | Hypothesis generation for new fields |

### 2.2 The Six Operators

1. **Combinatio** — Merge two disciplines in equal standing.
   *Biology + Computation → Bioinformatics*

2. **Subordinatio** — Apply the methods of discipline A to the objects of discipline B.
   *Methods of physics → biological objects → Biophysics*

3. **Mediatio** — Find a third discipline that bridges two distant ones.
   *Economics ↔ Neuroscience, mediated by Cognitive Psychology → Neuroeconomics*

4. **Amplificatio** — Extend a method native to a narrow field into broader domains.
   *Cladistics (biology) → applied to languages → Phylogenetic Linguistics*

5. **Quaestio** — Generate novel questions by transferring the fundamental question of one field to another.
   *"What symmetries exist?" (physics) → applied to social networks → Physics of Social Networks*

6. **Inversio** — Invert the fundamental question of one discipline and apply it to another.
   *"How do organisms adapt to environments?" inverted → "How do built environments adapt to organisms?" → Biophilic Design*

### 2.3 Plausibility Criteria

Not all combinations are fertile. A combination is plausible when:

- **Ontological overlap (partial):** The two disciplines share *some* but not *all* ontological assumptions about what entities exist and what causation means. Full overlap = same discipline. Zero overlap = likely vacuous.
- **Methodological transferability:** At least one method from discipline A can operate on the objects of discipline B without fundamental incompatibility.
- **Question novelty:** The combination generates questions that neither discipline asks independently.
- **Empirical grounding:** OpenAlex co-occurrence data shows either (a) emergent research activity (validating the combination) or (b) zero activity despite plausibility (indicating unexplored territory).

## 3. Data Architecture

### 3.1 Primary Data Source: OpenAlex

OpenAlex provides:
- **~65,000 concepts** in a hierarchical DAG (levels 0-5)
- **~250M+ works** with concept tags, citation data, and metadata
- **Co-occurrence data** via works tagged with multiple concepts
- **Temporal trends** via publication year filtering
- **Free API**, no authentication required for reasonable volumes

### 3.2 Taxonomy Layers

| Layer | Source | Size | Purpose |
|---|---|---|---|
| L0: Navigation | OECD FORD + custom curation | ~40 top-level fields | User-facing exploration |
| L1: Discipline | ASJC / OpenAlex level 1-2 | ~300 categories | Operator application |
| L2: Subdiscipline | OpenAlex level 2-3 | ~4,500 concepts | Enrichment & precision |
| L3: Concept | OpenAlex level 3-5 | ~65,000 concepts | Bibliometric validation |

## 4. System Architecture

```
┌─────────────────────────────────────────────────┐
│                 CONVERSATION LAYER               │
│         (Streamlit Chat / LLM Interface)         │
│  User selects disciplines, operators, explores   │
└─────────────────┬───────────────────────────────┘
                  │
┌─────────────────▼───────────────────────────────┐
│              COMBINATORIAL ENGINE                │
│  Applies 6 operators to discipline pairs/triples │
│  Generates hypotheses with justifications         │
│  Ranks by plausibility score                      │
└─────────────────┬───────────────────────────────┘
                  │
┌─────────────────▼───────────────────────────────┐
│            VALIDATION LAYER                      │
│  Queries OpenAlex for co-occurrence              │
│  Classifies: Established / Emergent / Virgin     │
│  Temporal trend analysis                          │
│  Literature sampling for evidence                 │
└─────────────────┬───────────────────────────────┘
                  │
┌─────────────────▼───────────────────────────────┐
│           KNOWLEDGE GRAPH                        │
│  NetworkX graph of disciplines                    │
│  Node properties: objects, methods, ontology      │
│  Edge weights: bibliometric co-occurrence         │
│  Persistent via JSON + Git versioning             │
└─────────────────┬───────────────────────────────┘
                  │
┌─────────────────▼───────────────────────────────┐
│           DATA INGESTION                         │
│  OpenAlex API bulk download                       │
│  LLM enrichment of node properties               │
│  Periodic refresh pipeline                        │
└─────────────────────────────────────────────────┘
```

## 8. Novel Contributions

1. **First complete implementation of Lullian combinatorics on the full map of sciences** with empirical validation.
2. **Six typed operators** (including Inversio, not in existing literature) that produce qualitatively distinct hypotheses.
3. **Plausibility scoring** based on ontological overlap — a new heuristic for evaluating interdisciplinary fertility.
4. **Territory classification** (Established / Emergent / Virgin / Implausible) backed by bibliometric evidence.
5. **Conversational interface** that makes systematic interdisciplinary exploration accessible to non-specialists.
